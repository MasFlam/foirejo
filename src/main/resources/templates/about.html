About Foirejo

Foirejo is an online marketplace for all legal products and services under the sun. All payments on the platform are made in Monero, a privacy preserving cryptocurrency. There are no fees on the platform, apart from the standard transaction fees that are required by the Monero network itself. Also, the entire project is free and open source software (see the GitHub repo) licensed under AGPLv3, which is great because it makes sure the users' software freedom is forever preserved, but it also means the project cannot apply for a CCS proposal, as the rules state that code written as part of a CCS proposal must be made available under a so-called "permissive" license.
